# 예스유학 — 시니어보딩 프렙 시스템

## 파일 구조
```
index.html          ← 메인 랜딩 페이지 (여기서 시작)
form/index.html     ← 학생 정보 입력 폼 (AI 분석)
proposal/eunhyu.html ← 박은휴 학생 제안서 샘플
guide/index.html    ← 입력 필드 가이드
README.md           ← 이 파일
```

## 배포 방법

### GitHub Pages (무료)
1. GitHub 저장소 생성 → 이 파일들 업로드
2. Settings → Pages → Source: main / / (root) → Save
3. https://[username].github.io/[repo-name] 자동 생성

### Netlify (무료)
- netlify.com → Deploy manually → 폴더 드래그 앤 드롭

### 로컬 실행
- index.html 을 브라우저로 열기

## AI 분석 설정
1. console.anthropic.com → API Keys → Create Key
2. 생성된 sk-ant-... 키를 폼 접속 시 입력
